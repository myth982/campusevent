# 🚀 QUICK START - Campus Event Hub

## Langkah Setup (SANGAT MUDAH)

### Step 1: Pastikan XAMPP Running ✅
```
1. Buka XAMPP Control Panel
2. Klik "Start" untuk Apache dan MySQL
3. Tunggu sampai berwarna hijau
```

### Step 2: Buka Install Wizard 🧙
```
URL: http://localhost/campusevent/install.php
```

### Step 3: Ikuti Wizard (3 Langkah) ⚡
```
1. Click "Lanjutkan" → Create Database
2. Click "Lanjutkan" → Import Schema  
3. Click "Lanjutkan" → Setup Password
4. Click "Selesai" → Buka Aplikasi
```

### Step 4: Login & Enjoy! 🎉
```
URL: http://localhost/campusevent

Email: mahasiswa@campus.com
Password: mhs123

atau

Email: admin@campus.com
Password: admin123
```

## ❓ Troubleshooting

### Error: "Unknown database 'campus_event_hub'"
✅ **Solusi:** Buka install.php, ikuti wizard setup otomatis

### Error: "Can't connect to MySQL"
✅ **Solusi:** Pastikan MySQL sudah running di XAMPP

### Error: "Permission denied"
✅ **Solusi:** Pastikan folder uploads/certificates/ exists dan writable
